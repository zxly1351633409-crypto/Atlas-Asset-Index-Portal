# 第五版内容登记与数据模型

更新时间：2026-07-14

## 内容规则

| 内容类型 | 在线内容 | 源文件 |
| --- | --- | --- |
| 视觉内容 | JPG、PNG 或 WebP 预览图 | PSD、AI 或辅助工程 |
| 模型 | 制作人员提供的渲染预览图 | BLEND、FBX、C4D 等工程 |
| UI | PNG 等界面预览图 | PSD、AI 或工具链接 |
| 文档需求 | DOCX 正文实时阅读 | DOCX 原文件 |
| 表格需求 | XLSX 工作表实时阅读 | XLSX 原文件 |

系统不解析 PSD 或 3D 工程。预览图存入 `preview_assets`，源文件存入 `source_files`，二者分别维护。

## 新增流程

1. 选择关联已有路径或提交本地文件。
2. 选择工作域、模块和版本，并可在三个层级现场新增。
3. 选择内容类型，表单自动切换预览和源文件字段。
4. 创建前检查必需的预览或需求文件。
5. 创建后标记完整、缺少预览、缺少源文件或路径待确认。

## 数据库结构

首个迁移包含以下数据表：

- `work_domains`
- `project_modules`
- `module_versions`
- `content_items`
- `preview_assets`
- `source_files`
- `content_relations`

当前页面仍使用模拟数据，数据库结构和迁移已经准备好。下一阶段接入本地索引服务后，再把新增记录和目录扫描结果写入数据库。
