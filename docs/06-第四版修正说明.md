# 第四版修正说明

更新时间：2026-07-14

## 本轮结论

- 需求内容主要来自文档或表格，原型使用 DOCX/XLSX，不再默认提供 PDF。
- 文档双击后实时呈现正文，表格双击后实时呈现工作表。
- 模型条目固定显示渲染预览图，双击查看大图；BLEND/FBX 等工程文件仍按需获取。
- 版本区域恢复横向时间基线、日期、版本节点和当前焦点，以便感知版本演进。
- 新增/关联内容时，可在工作域、模块、版本三个层级直接新增选项。

## 持久化边界

当前新增层级属于本地交互原型，能立即参与本次归档选择，但刷新后不会保留。SQLite 阶段需要把工作域、模块、版本和内容记录统一写入数据库，并补充重名校验、权限和删除规则。
